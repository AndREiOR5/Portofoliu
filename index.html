De ce apare „Run anyway”?

Windows poate afisa mesajul „Windows protected your PC” atunci cand rulezi aplicatia pentru prima data.

Acest lucru este normal deoarece aplicatia:

  • este creata de un dezvoltator independent
  • nu este semnata digital
  • este descarcata de pe internet

Pasii pentru a rula aplicatia:

  1. Apasa More info
  2. Apasa Run anyway
